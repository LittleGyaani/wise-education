== Description ==


Flipster is a CSS3 3D transform-based jQuery plugin built to replicate the familiar 'cover flow' effect, but also supports a variety of styles.	


== Installation ==

how to install


1. Upload the plugin files to the `/wp-content/plugins/flipster` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Flipster screen to configure the plugin
